# Бот системы лояльности магазина техники
#### Система лояльности розничного магазина техники на основе Телеграм бота с интеграцией 1С.
___
## Стек технологий
- Java + SpringBoot + Liquibase + Hibernate;
- TypeScript + React + Next.js;
- PostgreSQL;
- 1С.
___
